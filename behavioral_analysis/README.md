# Behavioral Analysis

## Overview

This module analyzes behavioral predictors of misinformation susceptibility using survey data, cognitive assessments, and statistical modeling.

## Components

- **notebooks/** - Exploratory data analysis, statistical testing, and visualization
- **scripts/** - Reusable Python modules for data processing and modeling
- **results/** - Statistical outputs, model summaries, and findings

## Key Analyses

- Cognitive vulnerability profiles
- Correlation with demographic factors
- Predictive modeling of susceptibility
- Exploratory factor analysis

## Data Sources

- IRB-approved survey responses
- Cognitive assessment instruments
- Behavioral metrics

## Technologies

- pandas, numpy, scipy for data processing
- scikit-learn for statistical modeling
- statsmodels for advanced statistical tests
- seaborn, matplotlib for visualization
