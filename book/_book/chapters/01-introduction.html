2  Introduction

2.1 Project Overview

The Tentacles of Misinformation is an integrated research and machine learning platform investigating cognitive and linguistic predictors of misinformation susceptibility.

2.1.1 Research Questions

  1. Who falls for misinformation? What behavioral traits predict vulnerability?
  2. What makes misinformation convincing? What linguistic patterns characterize fake news?
  3. Can we predict susceptibility? Can we build models that identify vulnerable individuals or content?
  4. How do we mitigate risk? What interventions reduce misinformation impact?

2.2 Scope and Timeline

This research expands on an IRB-approved Master’s thesis on cognitive vulnerability to misinformation, integrating:

  • Behavioral Analysis: Survey-based cognitive features (numeracy, critical thinking, conspiracy beliefs)
  • NLP Models: Deep learning classifiers for misinformation detection
  • Multimodal Fusion: Combining human behavior with content features
  • Interactive Dashboards: Decision support for policymakers and platforms

2.3 Methodology

2.3.1 Data Sources

  • Behavioral: Original survey data (anonymized, IRB-approved)
  • Content: FakeNewsNet dataset (Politifact + GossipCop)
  • Social: Twitter engagement metrics (where available)

2.3.2 Modeling Approach

  1. Phase 1 (Sprint 1): Data exploration + baseline classification
  2. Phase 2 (Sprint 2-3): Transformer-based NLP + feature fusion
  3. Phase 3 (Sprint 4-5): Multi-task learning + interpretability
  4. Phase 4: Research publication + open-source release

2.6 A Note on Ethics

Misinformation research sits at the intersection of cognitive science, platform design, and information security. This work acknowledges:

  • Platform incentives: Systems optimized for engagement often amplify misinformation
  • Cognitive realities: All humans have cognitive biases; vulnerability is not “stupidity”
  • Labeling challenges: “Fake news” is subjective; fact-checkers make judgment calls
  • Societal complexity: Misinformation thrives in contexts of economic inequality, social polarization, and institutional distrust

We aim to inform policy and product design without stigmatizing individuals or groups.

git clone https://github.com/sanjaykshetri/tentacles-of-misinformation.git

# Set up environment

conda activate misinformation

# Run data pipeline
python src/data_prep.py

# Render this book
cd book
quarto preview
